    b) Range based for loop like
    for(auto& x:vec) {

    }
c) Replaced NULL with nullptr.
d) Lambda expressions were introduced.
e) Smart pointers were introduced.
f) move semantics + rvalue reference.
g) constexpr
h) static asserts
i) override and final keywords 
j) default and delete
k) std::thread
l) variadic templates were introduced.

Que 3 - What is lvalue and rvalue?
Ans - An lvalue is something which has an address in the memory and it has a specific location which can be accessed repeatedly. 
An rvalue is something which is temporary and doesn't have a lasting location in memory. 

Qeu 4 - Why is move better then copying?
Ans - When we move we don't duplicate anything. We simple transfer ownership of one exisiting memory from one object to the other.
