Que 9 - What are some of the most important datastructures? What are their important features?
Ans - 
1) Array - Contiguous chunk of memory.
Cache locality - Excellent 
Access - O(1)
Search - O(n)
Insert at end - O(1)
Insert/delete in the middle - O(n)

2) Linked List
Poor cache locality
Access - O(n)
Insertion/deletion - O(1)

3) Stack
Cache - locality - It made using array or vector then good. If made using LL then bad. 
Implemented using array or linked List
Push - O(1)
Pop - O(1)
Top - O(1)

4) Queue 
Implemented using LL or array.
Push - O(1)
Pop - O(1)

Used in - BFS, scheduling, Buffering. 

5) Deque
Implemented using doubly linked list or cirular array. 
Push/pop front/back - O(1)

6) Hash map
Backend - Use hash table
Insert/search/delete - O(1)
Worst case - O(n)

7) Set
Usually balanced BST or hash table
Ordered set - Insert/search/delete - O(log n)
Unordered_set - Insert/search/Delete - O(1)

8) Heap/Priority Queue
Insert - O(log n)
Delete - O(log n)
Top - O(1)

9) BST
Insert/Delete/Seach - O(log n)

Ans - Ligthweight object introduced in C++11 through which we can initialise using curly braces.

This internally creates an intializer_list.

class A {

    public:

    vector<int> values;

    A(intializer_list<int> vals) {
