C++ memory management

Que 1 - What is fragmentation in Heap?
Ans - As the data gets deleted from the heap it creates a little segments of freed up memory and other memory locations are still occupied. This is called fragmentation. 
If a new memory of the size that is empty on heap is required then that gets allocated. Actually the heap keeps it empty for that case only.
If a larger chunk of memory is required on heap that is not avaialble on heap the heap mananger then manages the memory and reallocates the fragraments and the already occupied memory. This can take large time.
